Entity sync + resource viewing
Sync cards stay pinned left. Clicking a card's filename opens the entity file in the right pane's tab. The diff highlight (amber warning bar) shows the title mismatch inline. "viewing" badge on the selected card connects the two panes. User can check the file content before deciding which action to take.
The pattern
Left pane = workflow panel (pinned: extraction review, sync, reading list)
Right pane = tabbed content (graph preview, notes, files, entity files)

The right pane tabs work exactly like the existing content tab system. User opens references as needed — the workflow context on the left never gets lost. Clicking ⌘K can also open results directly in the right pane.
