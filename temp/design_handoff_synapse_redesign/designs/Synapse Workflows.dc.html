Agent-first development
A software development paradigm where AI agents write all code with zero manually-written code constraint. Pioneered by OpenAI's engineering team using Codex.
Key principles
• Repository as knowledge base — docs/ directory as system of record with progressive disclosure
• Agent legibility — all context accessible in-repo for agent understanding
• Architectural layering — Types → Config → Repo → Service → Runtime → UI
Observability
Uses observability stack with Victoria Logs, Metrics, and Traces with LogQL, PromQL, and TraceQL for monitoring agent activity.

Reading list + file viewing
Click "Reading List" in sidebar. Left pane shows items with tabs (pending/processing/ready). Selecting an item shows content preview on right.
